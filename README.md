# Running

## Installing Dependencies
#####`npm install`

####For running Grunt tasks:
#####`npm install -g grunt-cli`

## Running Tests

#####`grunt test`
   - Runs jshint and tests

## Serving

#####`grunt serve`
   - Runs jshint and tests
   - Starts up a server with nodemon and browserify
